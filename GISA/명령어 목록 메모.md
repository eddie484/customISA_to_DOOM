### 산술 명령어  
ADD    ; 덧셈  
SUB    ; 뺄셈  
MOV    ; 값 복사  
CMP    ; 두 값 비교해 nzcv 세팅 후 결과 버리기 (뺄셈 연산)  
MUL    ; 곱셈 후 하위 32비트 저장  
MULH   ; signed 곱셈 후 상위 32비트 저장  
MULHU  ; unsigned 곱셈 후 상위 32비트 저장  
DIV    ; signed 나눗셈 후 몫 저장  
DIVU   ; unsigned 나눗셈 후 몫 저장  
MOD    ; signed 나눗셈 후 나머지 저장  
MODU   ; unsigned 나눗셈 후 나머지 저장  

### shift 명령어
SHL    ; 왼쪽으로 shift  
ASR    ; 부호 유지하며 오른쪽으로 shift  
LSR    ; 0으로 채우며 오른쪽으로 shift  
ROL    ; 왼쪽으로 rotate  
ROR    ; 오른쪽으로 rotate  

### 논리 연산
AND    ; 비트 AND  
OR     ; 비트 OR  
XOR    ; 비트 XOR  
NOT    ; 비트 NOT  
BCHK   ; 두 값 비트 단위로 비교해 nzcv 세팅 후 결과 버리기 (AND 연산)  

### 메모리 연산  
LDR    ; 메모리에서 32비트 값 로드  
LDRB   ; 메모리에서 8비트 값 0으로 확장해 로드  
LDRSB  ; 메모리에서 8비트 값 부호 확장해 로드  
LDRH   ; 메모리에서 16비트 값 0으로 확장해 로드  
LDRSH  ; 메모리에서 16비트 값 부호 확장해 로드  
STR    ; 메모리에 32비트 값 저장  
STRB   ; 메모리에 하위 8비트 값 저장  
STRH   ; 메모리에 하위 16비트 값 저장  

### 컨트롤 연산
B      ; 조건 분기  
JMP    ; 무조건 점프  

### 시스템 명령어
SYSCALL  ; 시스템 콜  
RETEXC   ; 예외 처리 후 커널에서 유저로 복귀  
SETINT   ; 인터럽트 켜기  
CLRINT   ; 인터럽트 끄기  
WAITINT  ; 인터럽트 대기
LDRSYS   ; 시스템 레지스터 로드  
STRSYS   ; 시스템 레지스터에 저장

### 메모 

NOP: MOV r0, r0 으로 대체해 사용.

NEG: 
NOT r1, r0        ; 1의 보수
ADDI r1, r1, #1   ; 1을 더하면 2의 보수
로 대체해 사용.

32비트 즉시값 로드는 MOVI - SHL - ORI로 이용.

64비트 연산 많이 사용되게 된다면 캐리 포함 덧셈/뺄셈 향후 추가. 별로 안쓰면 

OS 구현중 필요할 경우 원자적 연산 처리 명령어 향후 추가

DOOM에 필요시 고정소수점 곱셈 연산 명령어 추가
